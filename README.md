# MinhaCritica
Aplicação para que usuários possam escrever críticas de livros.

## 🛠️ Funcionalidades
- Catalogo de livros;
- Buscar livros;
- Adicionar Livro;
- Adicionar uma critica a um livro;
- Editar e remover criticas;
- Visualizar criticas de usuarios;
